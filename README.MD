# CRUD Agile Webapp

# Quick Start Guide

1) **Clone Repo**
```
$ git clone 
$ cd Agile-CRUD-Webapp
```

2) **Create Virtual Envrionment**
```
$ python -m venv venv
$ source venv/bin/activate
```

3) **Install requirements**
```
$ pip install -r requirements.txt
```

4) **Launch Application**
```
$ flask run
```
